# Config Locations

## Alacritty

> $HOME/.config/alacritty/alacritty.yml

## NeoVim

> $HOME/.config/nvim/init.vim

## Tmux

> $HOME/.tmux.conf

## ZSH

> $HOME/.zshrc

## Starship

> $HOME/.config/starship.toml

## Pip

> $HOME/Library/Application Support/pip/pip.conf

## Sandboxd

> $HOME/.config/sandboxd/.sandboxrc
